# nvim-config
My Personal Neovim config written truly in lua

To Install:
  -  Make sure neovim and g++ and node is installed and are in `$PATH`
  -  For  linux/mac clone this repositry in `~/.config/nvim/lua/user`
  - For windows clone ths repo in  `~/AppData/Local/nvim/lua/user`
  - Rename your `init.lua` to `initbakup.lua`
  - Then add below in your init.lua
  ```lua
  require("user")
  ```
